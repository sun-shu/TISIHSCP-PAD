#### 项目流程

- 全局环境

> node版本 >= 18.19.1
>
> 全局安装 umi pnpm
>
>

> [项目地址](111.33.127.102:8090/tisihcsp/PROD01-TISIHCSP-EVA-MCS)

> git checkout main

- 构建依赖

> pnpm install

- 项目运行

> pnpm dev
